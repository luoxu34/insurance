## 儿童

### 投保顺序

国家少儿医保 → 意外险 → 重疾险 → 医疗险

其中，
* 重疾险：推荐配置保障至成年的消费型重疾险，成年后配置终身型保险
* 医疗险：预算充足时可考虑高保额免赔额的产品

### 意外险

1. 风险

   据教育部统计，目前 **意外伤害** 已成为我国 14 岁以下儿童的第一大死因。其中，溺水和交通意外的占比最大。

2. 防范

   2.1 通过读绘本等方式教育孩子辨别风险

   2.2 为孩子配置意外险
